# API_PYTHON

## Requisitos do projeto:

- Python 3.9
- Django 3.2.6
- djangorestframework 3.12.4
- drf-yasg 1.20.0

## Como executar o projeto:

- pip install -r requirements.txt
- python manage.py migrate
- python manage.py runserver
